# Next.js ISR PoC

# Server Node.js

```bash
    cd server

    npm install

    npm start
```

# Client Next.js

```bash
    yarn

    yarn build

    yarn start --port=3001
```

#### To test the revalidate, you can edit the countries in the page: <http://localhost:3001/admin>


#### To regenerate the countries database, you can use the seed: <http://localhost:3000/seed>
